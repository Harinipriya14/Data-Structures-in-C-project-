**E-Commerce Inventory Management System :**

      This is a simple C program developed using Data Structures.
      The project runs in a C compiler and helps manage products, customers, and orders in an e-commerce system using a menu-driven approach.

**Features :**

    Add products with quantity and price
    View available products
    Add customers
    Place orders
    Automatically reduce stock after order
    Runs completely in the compiler

**Data Structures Used :**

    Arrays – to store products, customers, and orders
    Structures (struct) – to group related data like product details and order details

**Technologies Used :** 

    Language: C
    Concepts: Arrays, Structures, Functions
    Compiler: GCC / MinGW
    IDE: VS Code or any C compiler

**How to Run (Compiler Only) :**

    1.Install VS Code or any C-supported IDE
    2.Install a C compiler (GCC / MinGW)
    3.Open the source file in the IDE
    4.Compile the program
